## [0.1.0] - 2026-07-10

Initial public release of `@redrixx/writ` and `@redrixx/writ-react`.

### Added

- Authority-oriented core with separate writer and reader capabilities.
- Strict entity existence lifecycle and named lenient operations.
- Synchronous persistence port and memory implementation.
- Strict composition-root registry.
- React adapter based on `useSyncExternalStore`.
- State-library recipes, evidence demo, package contract, and adoption guides.

### Pre-1.0 policy

While versions are `0.x`, minor releases may contain breaking API or behavioral
changes. Breaking changes will be called out here and in release notes. Pin an
exact version when stability is more important than receiving updates.
